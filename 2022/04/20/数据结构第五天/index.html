数据结构第五天

发表于 2022-04-20 更新于 2023-03-20 分类于 LeetCode

第九题： 有效的数独 请你判断一个  9 x 9 的数独是否有效。只需要 根据以下规则 ，验证已经填入的数字是否有效即可。

数字  1-9  在每一行只能出现一次。 数字  1-9  在每一列只能出现一次。 数字  1-9  在每一个以粗实线分隔的  3x3  宫内只能出现一次。（请参考示例图） 注意： 一个有效的数独（部分已被填充）不一定是可解的。 只需要根据以上规则，验证已经填入的数字是否有效即可。 空白格用  ‘.’  表示。

答案：

解题思路： 首先就是要将思路转换为三个条件，分别是求每一行是否有重复数字，每一列是否有重复数字，每一小块是否有重复数字。 那么如何求是否有重复数字呢，这里就通过对行和列分别创建二维数组来计算。首先对其创造全 0 矩阵，之后通过双循环，接下来就是最重要的一步。 charCodeAt()这个方法的目的是将数字转换为 Unicode 编码，index 是计数器，因为数独是 1-9 的数字，而这些 1-9 的数字想要放入 0-8 的矩阵中，就需要减 1 才能放进去。以每行数字举例，比如第一行数为["5","3",".",".","7",".",".",".","."],5 这个数就会放入rows[0][4]中，从而使rows[0][4]变为 1，即计数加 1。所以当如果这一行中出现第二个 5 时，就会使这里变为 2，所以可以通过判断计数器 index 是否大于 1 来判断数独是否有效。 对于三维数组来说，其结构是 3*3 共九个数组，以第一行为例，前 3 列 i 为 0，j 为 0、1、2 所以其位于[0][0][index]这个数组中，一直持续到第三行第三列即[2][2]都会位于[0][-][index]这个数组中，而判断条件和二维数组没有区别，最后整体做一个或操作即可。

第十题： 矩阵置 0 给定一个 m x n 的矩阵，如果一个元素为 0 ，则将其所在行和列的所有元素都设为 0 。

答案：

思路： 通过两个数组来分别存放布尔值，一开始都是 false，但是如果矩阵中存在某个值为 0，则将其行和列上的布尔值都变为 true。 之后再通过一组对比，用或比较，如果row[i]或column[j]中有一个为 true，则将其置 0。从而得到答案并返回。
