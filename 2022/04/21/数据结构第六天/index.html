数据结构第六天

发表于 2022-04-21 更新于 2023-03-20 分类于 LeetCode

第十一题：字符串中的第一个唯一字符 给定一个字符串 s ，找到 它的第一个不重复的字符，并返回它的索引 。如果不存在，则返回 -1 。

答案：

思路: 遍历字符串中的字符，如果这个字符串的首个出现位置和他最后一次出现位置相等，则证明其只出现唯一一次，而且因为indexof的特性，其从前向后计数，所以其是第一个字符。只要返回这个值所在的位置即可。 不存在则返回-1。 indexOf() –> 返回()中内容的首次出现位置 lastIndexOf() –> 返回()中内容的最后一次出现位置

第十二题：赎金信 给你两个字符串：ransomNote 和 magazine ，判断 ransomNote 能不能由 magazine 里面的字符构成。

如果可以，返回 true ；否则返回 false 。

magazine 中的每个字符只能在 ransomNote 中使用一次。

答案：

思路： 因为magazine中的字母只能被ransomnote使用一次，所以如果ransomnote的长度大于magazine的长度，就可以直接返回false，而如果magazine的长度大于等于ransomnote的长度，则需要判断其中每个字母出现的次数，所以设置了一个长度为26的数组来存放每个字母出现的次数，首先将magazine中出现的字母都存放进数组中，再将ransomnote需要的字母依次减去，如果小于0，则证明字母不够，可以返回false。
