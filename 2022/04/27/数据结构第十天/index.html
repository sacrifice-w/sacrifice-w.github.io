数据结构第十天

发表于 2022-04-27 更新于 2023-03-20 分类于 LeetCode

第十九题：有效的括号 给定一个只包括 ‘(‘，’)’，’{‘，’}’，’[‘，’]‘ 的字符串 s ，判断字符串是否有效。 有效字符串需满足：

  1. 左括号必须用相同类型的右括号闭合。
  2. 左括号必须以正确的顺序闭合。

答案：

思路： 这道题属于匹配问题，匹配问题很多都可以用栈来解决，而这道题的核心判别思路就是，怎么让两个相近的左右括号做匹配，因为左括号肯定先出现，所以每出现一个左括号，就将其推入栈。直到出现右括号，如果此时栈的空间为0或者是两个不符合规则，则返回false。如果栈空间不为0，则证明存在左括号，通过预存在哈希表中的括号值，如果这两个值相等，则表示匹配成功，将左括号出栈。

第二十题：用栈实现队列 请你仅使用两个栈实现先入先出队列。队列应当支持一般队列支持的所有操作（push、pop、peek、empty）：

实现 MyQueue 类：

void push(int x) 将元素 x 推到队列的末尾 int pop() 从队列的开头移除并返回元素 int peek() 返回队列开头的元素 boolean empty() 如果队列为空，返回 true ；否则，返回 false 说明： 你只能使用标准的栈操作——也就是只有 push to top, peek/pop from top, size, 和 is empty 操作是合法的。

答案：

思路： 用双栈来实现队列，其中一个作为入栈，一个作为出栈。 就是需要添加一个中间状态，将入栈的值放到出栈中，也就是将元素顺序反转，从而实现pop、peek操作。 这里需要注意的就是this和条件的判断，因为需要判断栈长度从而进行操作。
